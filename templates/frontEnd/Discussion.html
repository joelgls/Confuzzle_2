Desenvolvimento de um jogo educacional para ensinar programação para crianças.

Descrição: O objetivo deste projeto é criar um jogo educacional interativo e envolvente que introduza conceitos fundamentais de programação para crianças de forma acessível e divertida. O jogo será desenvolvido com uma abordagem lúdica, utilizando personagens e cenários cativantes que estimulem o interesse e o aprendizado das crianças.
